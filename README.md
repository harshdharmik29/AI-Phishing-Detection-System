# 🛡️ AI Phishing Detection System

An advanced AI-powered cybersecurity project that detects phishing URLs, emails, and malicious web content using Machine Learning and NLP.

## 🚀 Features
- URL-based phishing detection
- Email content analysis
- NLP-based suspicious text classification
- Real-time risk score
- Explainable AI output
- Dashboard for monitoring threats

## 🧠 AI/ML Techniques Used
- TF-IDF
- Logistic Regression
- Random Forest
- XGBoost
- BERT (for email text)
- Feature engineering for URLs

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Pandas
- Flask / FastAPI
- Streamlit
- HuggingFace Transformers
